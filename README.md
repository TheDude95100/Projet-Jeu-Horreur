# Projet-Jeu-Horreur
Un projet avec des coupains pour un jeu horreur basé sur une nouvelle écrite par Mahévyce

Jeu de base:
Survival Horror (classique), aucun moyen de se défendre

POV:
TPS (Exemple RE4, Silent Hill)

Grasphisme:
Réalisme

Mode linéaire?
Random niveau spawn Yokai?

Ambiance:
Musique/Sons d'ambiance/Lumière

Gameplay:
Déplacement 3eme personne
Caméra "vivante" (pov Yokai)
Sprint (avec endurance)
Se cacher
Vie? (ptdr tu te fais one shot)
Trouver des moyens de contrer les strums (du SEEEEEEEEL)

Pitch de base:
S'enfuir de l'école de mort

Clés? Enigmes?

Ecole Japonaise pre Seconde Guerre mondiale.

Multiple fins (1 good, 1 mauvaise)

Sors a l'aube ou crève. (Fin mauvaise; bye bye)
